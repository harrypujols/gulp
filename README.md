## Installation
- Install [node.js](http://nodejs.org)
- `$ npm install -g bower`
- `$ npm install -g gulp`

## Initial Setup

- `$ cd </path/to/this/folder>/gulp`
- `$ bower install`
- `$ npm install`

## Run Dev

- `$ gulp`
